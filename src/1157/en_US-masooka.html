Word Study

Statement

Given a word composed of both uppercase and lowercase alphabets, write a program to determine the most frequently used alphabet in the word. Note that the distinction between uppercase and lowercase should be ignored.

Input

The first line provides a word made up of uppercase and lowercase alphabets. The length of the given word will not exceed 1,000,000.

Output

On the first line, output the most frequently used alphabet in uppercase. If there are multiple alphabets with the same highest frequency, output ?.
