서로소 쌍의 개수

문제

네 정수 $a$, $b$, $c$, $d$가 주어졌을 때 $a \le x \le b$, $c \le y \le d$이고 최대공약수가 $1$인 순서쌍 $(x,\ y)$의 개수를 구하는 프로그램을 작성하시오.

입력

네 정수 $a$, $b$, $c$, $d$가 공백으로 구분되어 주어진다. ($1 \le a \le b \le 10\,000\,000$; $1 \le c \le d \le 10\,000\,000$)

출력

$a \le x \le b$, $c \le y \le d$이고 최대공약수가 $1$인 순서쌍 $(x,\ y)$의 개수를 출력한다.
