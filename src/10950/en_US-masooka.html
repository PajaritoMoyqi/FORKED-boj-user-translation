Statement

Write a program that takes two integers $A$ and $B$ as input, and then outputs $A + B$.

Input

The first line contains the number of test cases $T$.

Each test case consists of a single line, and on each line $A$ and $B$ are given. ($0 < A, B < 10$)

Output

For each test case, output $A + B$.
