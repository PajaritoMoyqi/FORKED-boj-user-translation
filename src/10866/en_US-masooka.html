Deque

Statement

Implement a Deque (Double-Ended Queue) for storing integers and write a program that processes commands given as input.

There are a total of eight commands:

  • push_front X: Insert the integer X at the front of the deque.
  • push_back X: Insert the integer X at the back of the deque.
  • pop_front: Remove and output the number at the front of the deque. If the deque is empty, output −1.
  • pop_back: Remove and output the number at the back of the deque. If the deque is empty, output −1.
  • size: Output the number of integers in the deque.
  • empty: Output 1 if the deque is empty, otherwise output 0.
  • front: Output the integer at the front of the deque. If the deque is empty, output −1.
  • back: Output the integer at the back of the deque. If the deque is empty, output −1.

Input

The first line contains the number of commands $N$ ($1 \le N \le 10\,000$) to be given. From the second line onwards, one command is given per line for $N$ lines. The integers provided are greater than or equal to 1 and less than or equal to 100,000. No command other than those listed in the problem will be given.

Output

For each command that requires an output, print the result on a new line.
