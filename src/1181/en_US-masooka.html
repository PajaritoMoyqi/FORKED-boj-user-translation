Word Sorting

Statement

Write a program to sort $N$ words composed of lowercase alphabets according to the following conditions:

  1. Sort by length in ascending order.
  2. If the lengths are the same, sort them in dictionary order.

Note that duplicate words should be removed, leaving only one instance of each.

Input

The first line contains the number of words $N$ ($1 \le N \le 20\,000$). From the second line onwards, one word composed of lowercase alphabets is given per line for a total of $N$ lines. The length of each word does not exceed 50 characters.

Output

Print the words sorted according to the conditions.
