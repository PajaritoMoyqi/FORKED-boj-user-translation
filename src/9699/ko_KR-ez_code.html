쌀 포대

문제

쌀 몇 포대를 다섯 고아원으로 보내려 합니다. 가장 무거운 포대는 가장 많은 고아가 있는 알-아민 고아원으로 보냅니다. 가장 가벼운 포대는 사는 아이가 가장 적은 무티아라 고아원으로 보냅니다. 쌀 포대가 여러 개 주어질 때, 어떤 포대가 알-아민 고아원으로 가야 하는지 계산할 수 있습니까?

입력

첫 줄에 테스트 케이스의 수를 나타내는 하나의 정수가 주어집니다. 이어지는 줄에는 5개 쌀 포대의 무게를 나타내는 5개 정수가 공백으로 구분되어 주어집니다. 포대의 무게는 100이 넘지 않습니다.

출력

각 테스트 케이스에 대해 Case #x: 이후 정수를 출력하는데, x는 (1부터 시작하는) 테스트 케이스의 번호를 나타내고 정수는 알-아민 고아원으로 가는 포대의 무게를 나타냅니다.
