Statistics

Statement

Handling numbers is a significantly important task in statistics. In statistics, the basic statistical values that represent $N$ numbers include the following, assuming that $N$ is an odd number:

  1. Arithmetic Mean: The sum of $N$ numbers divided by $N$
  2. Median: The middle value when $N$ numbers are arranged in increasing order
  3. Mode: The most frequently occurring number among the $N$ numbers
  4. Range: The difference between the maximum and minimum values among the $N$ numbers

Given $N$ numbers, write a program to calculate these four basic statistical values.

Input

The first line contains the number of values $N$ ($1 \le N \le 500\,000$), which is an odd number. The next $N$ lines contain integers. The absolute value of these integers does not exceed 4,000.

Output

On the first line, output the arithmetic mean. Round to the nearest integer.

On the second line, output the median value.

On the third line, output the mode. If there are multiple modes, output the second smallest value.

On the fourth line, output the range.

The arithmetic mean is $(0 + 0 + (-1)) / 3 = -0.333333\cdots$, which rounds to $0$. It should not be output as -0.
