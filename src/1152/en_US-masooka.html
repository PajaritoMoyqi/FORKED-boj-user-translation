Number of Words

Statement

A string composed of English uppercase and lowercase letters and spaces is given. How many words are in this string? Write a program to find this out. Note, if a word appears multiple times, it should be counted each time it appears.

Input

The first line gives a string made up of English uppercase and lowercase letters and spaces. The length of this string does not exceed 1,000,000. Words are separated by a single space, and there are no consecutive spaces. Also, the string can start or end with a space.

Output

On the first line, print the number of words.
