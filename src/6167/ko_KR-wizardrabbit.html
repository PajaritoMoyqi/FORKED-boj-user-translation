1, 2, 3 정렬

문제

길이 $N$의 수열 $A_1, A_2, \cdots A_N$이 주어진다. $A$의 각 원소의 값은 $1$, $2$, $3$ 중 하나이다.

당신은 아래의 연산을 $0$회 이상 수행할 수 있다.

  • $A$의 원소 하나를 고르고, 고른 원소의 값을 $1$, $2$, $3$ 중 하나로 바꾼다.

$A$를 비내림차순 또는 비오름차순으로 만들기 위해 사용해야 하는 연산의 최소 횟수는 몇 번일까?

입력

첫째 줄에 $N$이 주어진다. ($1 \le N \le 30\;000$)

둘째 줄에 $A_1, A_2, \cdots, A_N$이 공백을 사이에 두고 차례대로 주어진다. ($A_i \in \{1, 2, 3\}$; $1 \le i \le N$)

출력

문제의 정답을 출력한다.
