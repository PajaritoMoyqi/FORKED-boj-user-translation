Maximum Value

Statement

Write a program that, given 9 different natural numbers, finds the maximum value among them and determines which of the numbers it is.

For example, when given the following 9 different natural numbers,

3, 29, 38, 12, 57, 74, 40, 85, 61

the maximum value among them is 85, and it is the 8th number.

Input

From the first line to the ninth line, one natural number is given per line. Each natural number is less than 100.

Output

On the first line, print the maximum value. On the second line, print the position $k$ where the $k$-th number is the maximum.
