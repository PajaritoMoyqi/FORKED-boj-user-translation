Coordinate Sorting

Statement

Given $N$ points on a 2D plane, write a program that sorts and prints these points. The points should be sorted in increasing order of the x-coordinate, and in cases where the x-coordinates are the same, they should be sorted in increasing order of the y-coordinate.

Input

The first line contains the number of points $N$ ($1 \le N \le 100\,000$). From the second line to the $N$th line, the position of the $i$th point, $x_i$ and $y_i$, is given. ($-100\,000 \le x_i, y_i \le 100\,000$) The coordinates are always integers, and there are no two points with the same position.

Output

Print the sorted points, one point per line, starting from the first line.
