Number Search

Statement

Given $N$ integers A[1], A[2], …, A[$N$], write a program to determine if a certain integer $X$ exists within this array.

Input

The first line contains a natural number $N$ ($1 \le N \le 100\,000$). The next line contains $N$ integers A[1], A[2], …, A[$N$]. The following line contains $M$ ($1 \le M \le 100\,000$). The next line presents $M$ numbers, and the task is to find out if these numbers exist in the array A. The range of all integers is greater than or equal to $-2^{31}$ and less than $2^{31}$.

Output

Print the answer on $M$ lines. Print 1 if it exists, and 0 if it does not.
