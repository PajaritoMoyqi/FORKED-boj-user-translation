Maximum Rectangle

Description

Given a matrix of size NxM consisting of 1 and 0, write a program to find the largest rectangular submatrix that consists only of 1.

Input

The input consists of multiple test cases. The first line of each test case contains N and M. (1 ≤ N, M ≤ 1,000) The following N lines each contain M numbers separated by spaces. The last line contains two zeros. It means the end of the input.

Output

For each test case, print the area of the largest rectangle. If there is no such rectangle, print 0.
