배열 뒤집어서 출력하기

문제

정수로 이루어진 배열이 주어지면, 배열을 뒤집은 후 출력하는 프로그램을 작성해 보자.

예를 들어, $[3, 4, 2, 5, 1]$을 뒤집으면 $[1, 5, 2, 3, 4]$가 된다.

입력

입력은 여러 개의 테스트 케이스로 이루어져 있다.

각 테스트 케이스의 첫 번째 줄에는 배열의 크기 $N$이 주어진다. ($1 \le N \le 100$)

이후 $N$개의 줄에 걸쳐 배열의 원소가 순서대로 한 줄에 하나씩 주어진다. 이 때 배열에는 항상 $1$ 이상 $N$ 이하의 정수가 각각 한 번씩 등장한다.

입력의 끝에는 $0$이 주어지며, $0$이 입력으로 주어졌을 경우 여러분의 프로그램은 더 이상의 출력 없이 종료되어야 한다.

출력

각 테스트 케이스마다 배열을 뒤집은 결과를 출력하되, 배열의 원소들은 한 줄에 하나씩 순서대로 출력한다. 각 테스트 케이스의 출력이 끝날 때마다 그 다음 줄에 $0$을 출력한다.
