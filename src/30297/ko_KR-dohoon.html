Irreducible Permutation

문제

순열이 자신을 제외한 모든 접두사에 대해 순열을 이루지 않으면 '축소-불가능'하다. 예를 들어 $[2, 3, 1]$과 $[4, 1, 2, 3]$는 '축소-불가능'하고, $[2, 1, 3]$과 $[1 ,3, 2]$는 '축소-불가능'하지 않다.

길이가 $n$인 순열 $p$가 주어진다. 당신은 인접한 두 원소를 골라 교환할 수 있다.

$p$를 '축소-불가능'한 순열로 바꾸기 위해 필요한 교환 횟수의 최솟값과 그러한 교환 방법을 찾아라.

입력

첫 번째 줄에는 테스트 케이스의 수 $t$가 주어진다.

각 테스트 케이스의 첫 번째 줄에는 $n$, 두 번째 줄에는 $p_1, \ldots, p_n$이 주어진다.

출력

각 테스트 케이스마다 두 개의 줄에 걸쳐 다음과 같이 출력한다.

첫 번째 줄에는 $p$를 '축소-불가능'한 순열로 바꾸기 위한 교환 횟수의 최솟값, $k$를 출력한다.

두 번째 줄에는 $k$개의 공백으로 구분된 정수 $s_1, \ldots, s_k$ $(1 \le s_i \le n-1)$를 출력한다. 이는 순서대로 $i$번째 교환에서 $s_i$번째 원소와 $s_i+1$번째 원소를 골라 교환함을 의미한다.

가능한 답이 여러가지라면, 그중 아무 것이나 출력한다.

제한

  • $1 \le t \le 10^5$
  • $1 \le n \le 10^5$
  • $p$는 순열을 이룸이 보장됨.
  • 모든 테스트 케이스에 대한 $n$의 총합은 $5 \cdot 10^5$를 넘지 않음이 보장됨.
  • 입력으로 주어지는 모든 값은 정수임.
