Sorting Numbers 2

Statement

Given $N$ numbers, write a program to sort them in ascending order.

Input

The first line contains the number of numbers $N$ ($1 \le N \le 1\,000\,000$). In the next $N$ lines starting from the second line, the numbers are given. These numbers are integers whose absolute values are less than or equal to $1\,000\,000$. The numbers do not duplicate.

Output

Print the result in ascending order, one per line, in $N$ lines.
