최대합

문제

$n$개의 정수로 구성된 수열 a1, a2, …, an과 양의 정수 $k$ ($1 ≤ k ≤ n$)가 주어져 있다. 이때, 연속적으로 나열된 $k$개 정수의 합 Si = ai + ai+1 + … + ai+k−1 ($1 ≤ i ≤ n − k + 1$)의 최댓값을 출력하는 프로그램을 작성하시오.

입력

첫 번째 줄에 양의 정수 $n$ ($1 ≤ n ≤ 100000$)과 양의 정수 $k$ ($1 ≤ k ≤ n$)가 순서대로 공백으로 구분되어 주어진다. 두 번째 줄부터 $1 + i$ 번째 줄 ($1 ≤ i ≤ n$)까지는 수열의 $i$번째 항 ai ($−10000 ≤$ ai $≤ 10000$)가 주어진다.

출력

첫 번째 줄에 Si의 최댓값을 출력한다.
