NGE

Description

There is a sequence of size N, denoted as A = A1, A2, …, AN. For each element Ai, we want to find its Next Greater Element (NGE(i)). The NGE of Ai is defined as the leftmost number that is greater than Ai and is located to its right. If no such number exists, the NGE is -1.

For example, if A = [3, 5, 2, 7], then NGE(1) = 5, NGE(2) = 7, NGE(3) = 7, and NGE(4) = -1. In the case of A = [9, 5, 4, 8], we have NGE(1) = -1, NGE(2) = 8, NGE(3) = 8, and NGE(4) = -1.

Input

The first line of input contains an integer N (1 ≤ N ≤ 1,000,000), the size of the sequence A. The second line of input contains the elements of the sequence A, A1, A2, …, AN (1 ≤ Ai ≤ 1,000,000).

Output

The first and only line of output must contain N integers, NGE(1), NGE(2), …, NGE(N), separated by a space.
