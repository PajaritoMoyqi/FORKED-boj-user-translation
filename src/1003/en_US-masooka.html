Fibonacci Function

Statement

The following source is a C++ function that computes the $N$th Fibonacci number.

int fibonacci(int n) {
    if (n == 0) {
        printf("0");
        return 0;
    } else if (n == 1) {
        printf("1");
        return 1;
    } else {
        return fibonacci(n-1) + fibonacci(n-2);
    }
}

When fibonacci(3) is called, the following happens:

  • fibonacci(3) calls fibonacci(2) and fibonacci(1) (first call).
  • fibonacci(2) calls fibonacci(1) (second call) and fibonacci(0).
  • The second call to fibonacci(1) prints and returns 1.
  • fibonacci(0) prints and returns 0.
  • fibonacci(2) gets the results of fibonacci(1) and fibonacci(0), and returns 1.
  • The first call to fibonacci(1) prints and returns 1.
  • fibonacci(3) gets the results of fibonacci(2) and fibonacci(1), and returns 2.

1 is printed 2 times and 0 is printed 1 time. Given $N$, write a program to find out how many times 0 and 1 are printed when fibonacci(N) is called.

Input

The first line contains the number of test cases $T$.

Each test case consists of a single line containing $N$. $N$ is a natural number or 0, and is less than or equal to 40.

Output

For each test case, output the number of times 0 and 1 are printed, separated by a space.
