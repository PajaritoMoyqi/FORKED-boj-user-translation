NGF

Description

There is a sequence of size N, denoted as A = A1, A2, ..., AN. For each element Ai in the sequence, we want to find its Next Greater Frequency (NGF(i)).

Let F(Ai) denote the number of times Ai appears in the sequence A. The Next Greater Frequency (NGF) of Ai is defined as the leftmost number to the right of Ai that appears in the sequence A more times than F(Ai). If no such number exists, the NGF is -1.

For example, if A = [1, 1, 2, 3, 4, 2, 1], then F(1) = 3, F(2) = 2, F(3) = 1, and F(4) = 1. Since there is no number to the right of A1 that appears more than 3 times, NGF(1) = -1. In the case of A3, A7 is to the right and satisfies F(A3) = 2 < F(A7) = 1, so NGF(3) = 1. NGF(4) = 2, NGF(5) = 2, and NGF(6) = 1.

Input

The first line of input contains an integer N (1 ≤ N ≤ 1,000,000), the size of the sequence A. The second line of input contains the elements of the sequence A, A1, A2, ..., AN (1 ≤ Ai ≤ 1,000,000).

Output

The first and only line of output must contain N integers, NGF(1), NGF(2), ..., NGF(N), separated by a space.
