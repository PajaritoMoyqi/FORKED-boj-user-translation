Star Printing 2

Statement

Print stars in a pattern such that the first line contains 1 star, the second line contains 2 stars, and the $N$th line contains $N$ stars.

However, the stars should be aligned to the right (refer to the example).

Input

The first line will contain $N$ ($1 \leq N \leq 100$).

Output

Print the stars line by line, from the first line to the $N$th line, in order.
