시험 문제의 난이도

문제

바이토니 교수님은 $N$문제의 시험 문제의 난이도를 길이 $N$의 정수 수열 $A$로 구상하려 한다. 이 때, $A$는 아래의 조건을 모두 만족해야 한다.

  • $A$에는 $1$ 이상 $N$ 이하의 수가 각각 한 번씩 등장한다.
  • $A_i$와 $A_{i+1}$의 차는 $K$ 이상이다. ($1 \le i \le N - 1$)

시험 문제를 출제하느라 고생이 많으신 교수님을 위해 조교인 여러분이 수열 $A$를 만들어 보자.

입력

첫째 줄에 정수 $N$과 $K$가 공백으로 구분되어 주어진다. ($2 \le N \le 1\,000\,000$; $1 \le K \le N$)

출력

조건을 만족하는 $A$를 만들 수 있다면 $A_1, A_2, \cdots, A_N$를 공백으로 구분하여 출력한다. 가능한 경우가 여럿 존재한다면 그 중 아무거나 출력한다.

조건을 만족하는 $A$가 없다면, NIE를 출력한다.
