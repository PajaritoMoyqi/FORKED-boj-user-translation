알파벳 찾기

Statement

Given a word, S, composed only of lowercase alphabets, write a program that outputs the first occurrence position of each alphabet in the word. If the alphabet is not present in the word, output −1.

Input

The first line contains the word S. The length of the word does not exceed 100 and is made up only of lowercase alphabets.

Output

For each alphabet, output the position of the first occurrence of a, the first occurrence of b, …, and the first occurrence of z, separated by spaces.

If an alphabet is not present in the word, output −1. The first letter of the word is at position 0, and the second letter is at position 1.
