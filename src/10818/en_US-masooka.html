Minimum & Maximum

Statement

$N$ integers are given. Write a program to find the minimum and maximum values among them.

Input

The first line provides the number of integers $N$ ($1 \leq N \leq 1,000,000$). The second line provides $N$ integers separated by spaces. All integers are greater than or equal to $-1,000,000$ and less than or equal to $1,000,000$.

Output

On the first line, print the minimum and maximum values among the $N$ integers given, separated by a space.
