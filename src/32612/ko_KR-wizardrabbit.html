주사위의 눈의 수의 합의 기댓값

문제

한별이는 주사위를 $N$개 던졌다. 각 주사위의 면의 수는 $X_1, X_2, \cdots, X_N$이다. 면의 수가 $X$개인 주사위는 각 면의 눈의 수가 $1, 2, \cdots, X$개이며, 각 면이 나올 확률은 동일하다.

이 주사위들을 모두 던졌을 때 나오는 주사위의 눈의 수의 합의 기댓값을 구하여라.

입력

첫째 줄에 $N$이 주어진다. ($1 \le N \le 8$)

둘째 줄에 $X_1, X_2, \cdots, X_N$이 공백으로 구분되어 주어진다. ($2 \le X_i \le 8$; $1 \le i \le N$)

출력

첫째 줄에 주사위들을 모두 던졌을 때 나오는 주사위의 눈의 수의 합의 기댓값을 출력한다. 절대/상대 오차는 $10^{-6}$까지 허용한다.
