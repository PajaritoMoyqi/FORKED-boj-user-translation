비타로의 관광 여행

문제

JOI 도시에는 수직선상의 긴 도로가 있습니다. 이 도로 위의 각 위치는 정수 좌표로 나타낼 수 있습니다. 도시에는 $1$부터 $N$까지의 번호가 붙은 $N$개의 관광지가 있으며, $i$번째 관광지의 좌표는 $X_i$로 주어집니다.

비타로는 JOI 도시의 모든 관광지를 방문하려 합니다. 그의 인생 모토는 "그리디"이기 때문에, 그는 다음의 절차를 따라 관광지를 방문하려 합니다:

  • 비타로의 현재 위치를 $x$라 합시다.
  • 아직 방문하지 않은 관광지 중에서, 비타로의 현재 위치로부터 거리 $|x - X_i|$가 최소인 관광지 $i$를 선택합니다. ($|t|$는 $t$의 절댓값을 나타냅니다.)
  • 비타로는 해당 관광지로 이동하여 방문합니다. 만약 동일한 거리에 여러 관광지가 있다면, 좌표가 가장 작은 관광지를 선택합니다.

비타로는 이런 방식으로 이동하면 생각보다 긴 거리를 이동하게 될 것이라는 것을 알고 있습니다. 시작 위치에 따라 이동 거리가 달라지기 때문에, 비타로는 $Q$개의 시작 후보 위치 $S_1, S_2, \cdots, S_Q$에서 시작했을 때 각각 얼마나 많은 거리를 이동해야하는지 알고 싶어합니다.

당신의 임무는 주어진 JOI 도시와 시작 위치들의 정보를 토대로, 각 시작 위치에서의 총 이동 거리를 계산하는 프로그램을 작성하는 것입니다.

입력

다음과 같은 표준 입력이 들어옵니다.

$N$

$X_1$ $X_2$ $\cdots$ $X_N$

$Q$

$S_1$

$S_2$

$\vdots$

$S_Q$

출력

$Q$개의 줄로 표준출력합니다. $j$번째 ($1\le j \le Q$) 줄에는 비타로가 좌표 $S_j$에서 출발해야할 때 총 이동 거리를 출력해야 합니다.

제한

  • $1 ≤ N ≤ 200\,000$.
  • $1 ≤ Q ≤ 200\,000$.
  • $0 ≤ X_i ≤ 10^9$ ($1 ≤ i ≤ N$).
  • $X_i < X_{i+1}$ ($1 ≤ i ≤ N - 1$).
  • $0 ≤ S_j ≤ 10^9$ ($1 ≤ j ≤ Q$).
  • 입력으로 주어지는 수는 모두 정수입니다.

서브태스크

번호 배점 제한
1 5

$Q = 1$, $N ≤ 2\,000$.
2 10

$Q = 1$.
3 30

$X_{i+1} − X_i ≤ 100$ ($1 ≤ i ≤ N - 1$).
4 55

추가 제약 조건 없음

비타로가 좌표 $7$에서 시작하여 관광지를 방문하는 예는 다음과 같습니다:

  1. 그는 아직 관광지 $1, 2, 3, 4, 5$를 방문하지 않았으며 비타로의 현재 위치에서의 거리는 각각 $7, 2, 1, 0, 2$입니다. 관광지 $4$는 비타로로부터 가장 가까운 관광지이기 때문에 그는 좌표 $7$에 머물며 관광지 $4$를 방문합니다.
  2. 그는 아직 관광지 $1, 2, 3, 5$를 방문하지 않았으며 비타로의 현재 위치에서의 거리는 각각 $7, 2, 1, 2$입니다. 관광지 $3$은 비타로로부터 가장 가까운 관광지이기 때문에 그는 좌표 $7$에서 좌표 $6$으로 이동하고 관광지 $3$을 방문합니다.
  3. 그는 아직 관광지 $1, 2, 5$를 방문하지 않았으며 비타로의 현재 위치에서의 거리는 각각 $6, 1, 3$입니다. 관광지 $2$는 비타로로부터 가장 가까운 관광지이기 때문에 그는 좌표 $6$에서 좌표 $5$로 이동하고 관광지 $2$를 방문합니다.
  4. 그는 아직 관광지 $1, 5$를 방문하지 않았으며 비타로의 현재 위치에서의 거리는 각각 $5, 4$입니다. 관광지 $5$는 비타로로부터 가장 가까운 관광지이기 때문에 그는 좌표 $5$에서 좌표 $9$로 이동하고 관광지 $5$를 방문합니다.
  5. 그는 아직 관광지 $1$만 방문하지 않았습니다. 관광지 $1$은 비타로로부터 가장 가까운 관광지이기 때문에 그는 좌표 $9$에서 좌표 $0$으로 이동하고 관광지 $1$을 방문합니다.

비타로의 총 이동 거리는 $15$이므로 $15$를 출력합니다.

이 샘플 입력은 모든 서브태스크의 제약 조건을 만족합니다.

이 예제 입력은 서브태스크 $3, 4$를 만족합니다.
