Sorting Numbers 3

Statement

Write a program to sort $N$ numbers in ascending order.

Input

The first line contains the number of numbers $N$ ($1 \le N \le 10\,000\,000$). From the second line, $N$ numbers are given. These numbers are natural numbers less than or equal to 10,000.

Output

Print the sorted numbers in ascending order, one per line, starting from the first line.
