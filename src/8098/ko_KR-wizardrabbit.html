강의 최대한 오래 듣기

문제

앨리스는 강의 듣는 것을 좋아하는 대학생이다. 앨리스가 다니는 학교에는 강의가 총 $N$개 있으며, $i$번째 강의는 시작하는 시간 $P_i$와 끝나는 시간 $K_i$가 있다.

앨리스는 최대한 오랫동안 강의를 듣고 싶어하지만, 몇몇 강의들은 일정이 서로 겹칠 수 있기 때문에 듣지 못하는 강의가 생길 수 있다. 단, 어떤 강의가 끝나자마자 새로운 강의가 시작하는 경우, 두 강의는 겹치지 않는다고 가정한다. 또한 어떤 강의의 시작 시간이 $a$이고 끝나는 시간이 $b$라면, 해당 강의의 수업 시간은 $b - a$로 계산한다.

앨리스가 일정이 겹치지 않도록 강의를 들을 때, 앨리스가 들을 수 있는 총 수업 시간의 최댓값을 구해 보자!

입력

첫째 줄에 $N$이 주어진다. ($1 \le N \le 10\;000$)

둘째 줄부터 총 $N$줄에 걸쳐 각 강의의 정보가 한 줄에 하나씩 주어진다. $i$번째 강의의 정보에서는 $P_i$, $K_i$가 공백으로 구분되어 주어진다. ($0 \le P_i \lt K_i \le 30\;000$; $P_i$, $K_i$는 정수)

출력

앨리스가 들을 수 있는 총 수업 시간의 최댓값을 출력한다.
