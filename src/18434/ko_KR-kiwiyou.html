올림픽 버스

문제

JOI 왕국에는 1부터 N까지의 번호가 붙은 N개의 도시와 도시들을 연견하는 1번부터 M번까지의 M개 버스 노선이 있다. i번 버스 노선은 Ui번 도시에서 Vi번 도시로 향하며, 운임은 Ci이다. i번 버스 노선에서 승객은 도시 Ui에서만 탑승할 수 있고, Vi에서만 하차할 수 있다. 한 도시에서 다른 도시로 가는 버스 노선은 하나 이상일 수 있다.

올림픽이 JOI 왕국에서 개최될 예정이다. K는 JOI 왕국의 교통부 장관이다. K는 올림픽이 열리기 직전 최대 하나의 버스 노선을 골라 운임은 그대로 두고 노선의 방향을 뒤집을 수 있다. 다르게 말해서, i번 버스 노선을 골라 Ui에서 Vi로 향하는 것이 아니라 Vi에서 Ui로 향하도록 할 수 있다. (1 ≤ i ≤ M) 노선 방향을 뒤집는 데 드는 비용은 Di이고, K가 지불할 것이다. 혼란을 막기 위해 올림픽 중에는 노선의 방향을 뒤집을 수 없다.

K는 교통부 장관이므로 올림픽 중에 버스를 타고 1번 도시와 N번 도시를 왕복할 예정이다. K는 적당한 노선의 방향을 바꿔 왕복 운임에 노선의 방향을 바꾸는 비용을 더한 값을 최소로 하려고 한다.

도시의 수와 버스 노선의 정보가 주어지면 왕복 운임과 노선의 방향을 바꾸는 비용의 합의 최솟값을 구하자. 1번 도시와 N번 도시를 왕복할 수 없다면 -1을 대신 출력한다.

입력

표준 입력에서 아래 데이터를 읽는다. 주어지는 수는 모두 정수이다.

N M
U1 V1 C1 D1
⋮
UM VM CM DM

출력

표준 출력에 왕복 운임과 노선의 방향을 바꾸는 비용의 합의 최솟값을 출력한다. 만약 1번 도시와 N번 도시를 왕복할 수 없다면 -1을 대신 출력한다.

제한

  • 2 ≤ N ≤ 200.
  • 1 ≤ M ≤ 50 000.
  • 1 ≤ Ui ≤ N (1 ≤ i ≤ M).
  • 1 ≤ Vi ≤ N (1 ≤ i ≤ M).
  • Ui ≠ Vi (1 ≤ i ≤ M).
  • 0 ≤ Ci ≤ 1 000 000 (1 ≤ i ≤ M).
  • 0 ≤ Di ≤ 1 000 000 000 (1 ≤ i ≤ M).

서브태스크

번호 배점 제한
1 5

M ≤ 1000.
2 11

M은 짝수, U2i−1 = U2i, V2i−1 = V2i, C2i−1 = C2i (1 ≤ i ≤ M/2).

3 21

4 63

추가 제약 조건 없음.

K가 2번 버스 노선의 방향을 바꾸면 드는 비용은 1이다. 이때 1번 도시에서 4번 도시로 가는 최소 운임은 6이고, 4번 도시에서 1번 도시로 가는 최소 운임은 3이다. 따라서 1번 도시와 4번 도시를 왕복하는 운임과 노선 방향을 바꾸는 비용의 합은 10이다.

1번 도시와 4번 도시를 왕복하는 운임과 노선 방향을 바꾸는 비용의 합은 10보다 작아질 수 없으므로 10을 출력한다.
