Stack

Statement

Implement a stack that stores integers, and write a program that processes commands given as input.

There are a total of five commands:

  • push $X$: A command to push the integer $X$ onto the stack.
  • pop: Remove the integer at the top of the stack and output it. If there are no integers in the stack, output -1.
  • size: Output the number of integers in the stack.
  • empty: If the stack is empty, output 1; otherwise, output 0.
  • top: Output the integer at the top of the stack. If there are no integers in the stack, output -1.

Input

The first line contains the number of commands $N$ ($1 \le N \le 10\,000$) to be given. From the second line onwards, one command is given per line. The given integers are greater than or equal to 1, and less than or equal to 100,000. No command other than those listed in the problem will be given.

Output

For each command that requires an output, print the result on a new line.
