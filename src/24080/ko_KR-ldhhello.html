복잡한 문자열

문제

글자를 처음 배우는 아기 동현이는 아직 너무 어려서 복잡한 문자열은 읽지 못한다.

동현이는 문자열을 이루는 글자가 세 종류 이상이면 너무 복잡해서 읽지 못한다고 한다.

아기 동현이를 위해 A, B, C, D, E 로 구성된 문자열 $S$가 주어졌을 때, 이 문자열이 복잡한 문자열인지 판단하는 프로그램을 작성해 주자!

입력

첫 번째 줄에 정수 $N$이 주어진다.

두 번째 줄에 길이 $N$의 문자열 $S$가 주어진다.

출력

입력된 문자열 $S$가 복잡한 문자열이라면 Yes를, 아니라면 No를 출력한다.

제한

  • $1 \le N \le 100$
  • 문자열 $S$의 모든 문자는 A, B, C, D, E 중 하나이다.
  • A, B, E의 세 종류의 문자가 등장한다. 따라서 주어진 문자열은 복잡한 문자열이므로 Yes를 출력한다.
  • A, D의 두 종류의 문자가 등장한다. 따라서 주어진 문자열은 복잡한 문자열이 아니므로 No를 출력한다.
