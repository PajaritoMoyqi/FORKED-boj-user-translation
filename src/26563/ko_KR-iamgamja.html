시험

문제

친구와 당신은 $n$개의 O/X문제로 구성된 시험을 봤습니다. 당신은 당신이 제출한 답안과 친구가 제출한 답안을 알고 있습니다. 또 친구가 $k$문제를 맞았다는 것을 알고 있습니다.

당신이 맞을 수 있는 문제 개수의 최댓값을 계산하세요.

입력

첫째 줄에는 테스트케이스의 수 m이 주어집니다.

각 테스트케이스는 다음과 같이 구성되어 있습니다.

  • 첫째 줄에는 정수 $k$ ($0 ≤ k ≤ n$)가 주어집니다.
  • 둘째 줄에는 당신이 제출한 답안을 나타내는 길이 $n$ ($1 ≤ n ≤ 1000$)의 문자열이 주어집니다. 문자열은 T, F로만 이루어져 있습니다.
  • 셋째 줄에는 친구가 제출한 답안을 나타내는 길이 $n$ ($1 ≤ n ≤ 1000$)의 문자열이 주어집니다. 문자열은 T, F로만 이루어져 있습니다.

출력

각 테스트케이스마다, 당신이 맞을 수 있는 문제 개수의 최댓값을 출력하세요.
