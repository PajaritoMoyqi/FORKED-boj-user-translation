$2n$회보다 많이 등장하는 수

문제

$50n$개의 수가 한 줄에 다섯 개씩 주어진다. $2n$회보다 많이 등장하는 수들을 찾아보자.

입력

첫째 줄에 정수 $n$이 주어진다. ($1 \le n \le 1\,000$)

이후 $10n$개 줄에 걸쳐, 각 줄에 서로 다른 다섯 개의 수가 공백으로 구분되어 주어진다. 각 수는 $1$ 이상 $50$ 이하의 정수이다.

출력

$2n$회보다 많이 등장하는 수들이 있다면, 첫째 줄에 해당하는 수들을 공백으로 구분하여 오름차순으로 출력한다.

$2n$회보다 많이 등장하는 수들이 하나도 없다면, 첫째 줄에 $-1$을 출력한다.
