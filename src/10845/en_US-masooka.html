Queue

Statement

Implement a queue that stores integers, and write a program that processes commands given as input.

There are six types of commands:

  • push $X$: An operation that puts the integer $X$ into the queue.
  • pop: Removes the integer at the front of the queue and prints it. If there are no integers in the queue, print −1.
  • size: Prints the number of integers in the queue.
  • empty: If the queue is empty, print 1; otherwise, print 0.
  • front: Prints the integer at the front of the queue. If there are no integers in the queue, print −1.
  • back: Prints the integer at the back of the queue. If there are no integers in the queue, print −1.

Input

The first line gives the number of commands $N$ ($1 \le N \le 10\,000$). From the second line, one command is given per line for $N$ lines. The given integers are greater than or equal to 1 and less than or equal to 100,000. There are no commands other than those mentioned in the problem.

Output

For each command that requires output, print one response per line.
