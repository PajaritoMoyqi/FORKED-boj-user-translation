거듭제곱 문제

문제

길이 $N$의 양의 정수로만 이루어진 수열 $A_1, A_2, \cdots, A_N$이 주어지면, ${A_1}^2 + {A_2}^3 + \cdots + {A_N}^{N + 1}$을 양의 정수 $M$으로 나눈 나머지를 출력하는 프로그램을 작성하시오.

입력

첫째 줄에 $N$, $M$이 공백을 사이에 두고 차례대로 주어진다.

둘째 줄에 $A_1, A_2, ..., A_N$이 공백을 사이에 두고 차례대로 주어진다.

출력

문제의 정답을 출력한다.

제한

  • $1 \le N \le 10^6$
  • $1 \le M \le 10^7$
  • $1 \le A_i \le 10^5$ ($1 \le i \le N$)
