Statement

Write a program that reads two integers $A$ and $B$ and then prints out $A + B$.

Input

The input consists of several test cases.

Each test case is on a single line, which contains $A$ and $B$. ($0 \lt A, B \lt 10$)

The end of input is indicated by two zeros.

Output

Print out $A + B$ for each test case.
