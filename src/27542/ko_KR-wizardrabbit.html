절댓값 수열

문제

길이 $N$의 수열 $A_1, A_2, \cdots, A_N$가 주어진다. $A$에 아래의 연산을 $N - 1$회 적용한 후, $A_1$의 값을 출력하여라.

  • 연산 전 $A$의 크기를 $X$라 했을 때, $A_1, A_2, \cdots, A_X$를 $|A_1 - A_2|, |A_2 - A_3|, \cdots, |A_{X - 1} - A_X|$으로 변경한다. 즉, 연산 이후 수열의 크기는 $1$ 감소한다.

입력

첫째 줄에 $N$이 주어진다.

둘째 줄에 $A_1, A_2, \cdots, A_N$가 주어진다.

출력

문제의 정답을 출력한다.

제한

  • $2 \le N \le 2\,000$
  • $0 \le A_i \le 10^9$ ($1 \le i \le N$)
  • 입력으로 주어지는 모든 수는 정수

서브태스크

번호 배점 제한
1 25

$N = 2$
2 75

추가 제한 조건 없음

처음에는 $A = [3, 1, 4, 1]$인 상태이다. $A$의 길이는 $4$이므로 총 $3$회의 연산을 적용해야 한다.

  • 첫 번째 연산을 진행하면 $[3, 1, 4, 1]$은 $[2, 3, 3]$이 된다.
  • 두 번째 연산을 진행하면 $[2, 3, 3]$은 $[1, 0]$이 된다.
  • 세 번째 연산을 진행하면 $[1, 0]$은 $[1]$이 된다.

연산을 $3$회 진행한 후 $A_1 = 1$이므로, $1$을 출력하면 정답이다.
