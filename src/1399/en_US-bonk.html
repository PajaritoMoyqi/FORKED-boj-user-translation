Treasure Location

Statement

Define a function $\text{dig}$ as follows:

  • $\text{dig}(x) = x$ for $(0 \leq x \leq 9)$
  • $\text{dig}(x) = \text{dig}($sum of all digits of $x)$  $(x \geq 10)$

For example, $\text{dig}(49) = \text{dig}(13) = \text{dig}(4) = 4$

Omin has a very old map and is looking for treasure following a pirate's instructions. The old map outlines how to find the treasure.

Currently, Omin is facing north, and his current position is $(0, 0)$. North is the direction where the $Y$-coordinate increases, and east is the direction where the $X$-coordinate increases.

Omin can repeat the following steps $K$ times to find the treasure's location. The gold number starts at $1$

  1. Move forward by the value of $\text{dig(gold number)}$. Then, turn $90$ degrees clockwise.
  2. Multiply the gold number by $M$.

Write a program to output Omin's final location, i.e., the treasure's location.

Input

The first line contains the number of test cases, $T$. Each test case consists of two integers, $K$ $(0 \leq K \leq 10^9)$ and $M$ $(0 \leq M \leq 1\,000)$.

Output

p>For each test case, output the treasure's location in the format X Y.
