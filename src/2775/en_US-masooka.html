I Want to be the Women's Association President

Statement

Judy, who enjoys participating in community meetings, wants to become the women's association president this time and is planning to host a meeting by gathering people from each floor.

To reside in this apartment complex, there is a condition: To live in the $a$th floor, $b$th apartment, you must bring as many people as the sum of the number of people from the 1st to the $b$th apartment on the floor below ($a-1$). This contract condition must be strictly followed.

Assuming that there are no empty apartments in the complex and all residents have complied with this contract condition, output how many people live in the $k$th floor, $n$th apartment, given the positive integers $k$ and $n$. Note that the apartment complex has floors starting from 0, and each floor has apartments starting from 1. In the $i$th apartment of the 0th floor, $i$ people live.

Input

The first line is given the number of test cases $T$. For each test case, the first line is given an integer $k$, and the second line an integer $n$.

Output

For each test case, output the number of residents living in the specified apartment.

Constraints

  • $1 \le k, n \le 14$
