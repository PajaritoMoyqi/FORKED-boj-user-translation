Counting Digits

Statement

Given three natural numbers A, B, and C, write a program that calculates the product A × B × C and determines how many times each digit from 0 to 9 is used in the result.

For example, if A = 150, B = 266, and C = 427, then A × B × C = 150 × 266 × 427 = 17037300. In the result 17037300, the digit 0 is used 3 times, the digit 1 is used once, the digit 3 is used twice, and the digit 7 is used twice.

Input

The first line contains the number A. The second line contains the number B. The third line contains the number C. All numbers A, B, and C are natural numbers greater than or equal to 100 and less than 1,000.

Output

The first line should print the number of times the digit 0 appears in the product A × B × C. From the second to the tenth line, print how many times the digits from 1 to 9 appear in the product A × B × C, respectively. Each count should be printed on a new line.
